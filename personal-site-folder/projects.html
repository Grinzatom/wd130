Projects

Look at some of our practical projects. You can achieve some of them in a couple of hours by following the assembly and code step by step.

LCD display project

In this project we see how to connect and transmit images to an LCD screen. For this we use an Arduino Uno, a 1602A screen, a 220ohm resistor, a potentiometer and some jumpers. I did this project once with a 16-year-old boy for his science class and it took us around 30 minutes to assemble and about 15 minutes to program and test. In the attached image I have recreated the same setup for you

Distance ultrasom sensor

In this project you will learn how to use an ultrasound sensor. It has multiple uses as a mini radar. control liquid level or measure distances. For this we use an HC-SR04 sensor with an Arduino Uno and connectors as well as an alarm speaker indicating when the programmed distance is reached.
